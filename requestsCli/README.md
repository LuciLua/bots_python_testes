## How to start

### 1. Install python packages
```bash
pip install -r .\requirements.txt
```

### 2. Execute the code

```bash
python .\requestCli.py [...params] 
```

#### Params:
```
-h help 

-m method

-p path*

-v value

-k key

-H header

-c cookies 

-C content 

-cp copy site

-o open browser after copysite

-s save content in a json file
```
*Must have
